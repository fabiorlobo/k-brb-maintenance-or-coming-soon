BRB creates a very simple maintenance mode / coming soon page for your site.

== Description ==

Your site is under construction or maintenance? Then say to your visitors that you will Be Right Back!

BRB creates a very simple maintenance mode / coming soon page for your site. You can put your logo on it, write a text and link your social media. Also, BRB let you define what kind of user is capable to see the site.

You can easily customize the colors of the page â€“ also, you can put HTML, JS or even CSS on it.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/k-brb` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Maintenance / Coming Soon screen to configure the plugin

== Frequently Asked Questions ==

= Is it possible to reset settings to defaults? =

Yes! Just deactivate BRB in Plugins menu.
